# xml2doc — Project Documentation

This folder contains structured project documentation for **xml2doc**, organized following [arc42](https://arc42.org) concepts and [C4 model](https://c4model.com) architecture views.

## Purpose

xml2doc converts .NET XML documentation files (generated by the C# compiler) into deterministic, linkable Markdown. It is consumed by any .NET developer who wants API documentation in Markdown format, either as a CLI tool, an MSBuild integration, or via the Core library.
