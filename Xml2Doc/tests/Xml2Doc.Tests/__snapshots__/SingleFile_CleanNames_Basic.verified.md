# Mathx

Math helpers for demos.

## Method: Int32)
- `Method: Int32)`
Add three integers.

**Parameters**
- `a` — First.
- `b` — Second.
- `c` — Third.

**Returns**

Sum of all.

- `Method: Int32)`
Add two integers.

**Parameters**
- `a` — Left addend.
- `b` — Right addend.

**Returns**

Sum of `a` and `b`.

**Example**

```csharp
var s = Mathx.Add(1,2); // 3
```
