# Mathx

Math helpers for demos.

<a id="xml2doc.sample.mathx.add(int,int,int)"></a>

## Method: Add(int, int, int)

Add three integers.

**Parameters**

- `a` — First.
- `b` — Second.
- `c` — Third.

**Returns**

Sum of all.

<a id="xml2doc.sample.mathx.add(int,int)"></a>

## Method: Add(int, int)

Add two integers.

**Parameters**

- `a` — Left addend.
- `b` — Right addend.

**Returns**

Sum of `a` and `b`.

**Example**

```csharp
var s = Mathx.Add(1,2); // 3
```
